Debugging with LLDB

Posted on Aug 2, 2021

记录 lldb 常用命令

断点

b main
breakpoint delete/disable/enable
b xxx.cpp : 47
r  # 开始执行
c  # stop 到 下一个断点

栈帧

bt    # 展示栈帧
f 0   # 切换到0号栈帧
f     # 定位到当前正在执行的程序段
fr v  # 列出当前栈帧/函数内所有变量
l 47  # 查看行数代码

查看变量

p a                   # 查看变量 a
w s v xxx             # watch 变量xxx，若有改变，continue 时 stop
w s v -w read  xxx    # read 时也stop
